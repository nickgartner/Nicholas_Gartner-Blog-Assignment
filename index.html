Styling Content.

Divs and Spans

Divs are containers that are used to create layouts, by putting html content inside of themand then altering the properties of the container.

Classes and ids are used to add specificity to divs as well as the content inside of the divs allowing for the control of the container as well as controlling the contentwithin the div.

Divs are transformed by adding padding and margins to them:

Margins are the empty space that surrounds an element.

Padding is the empty space that surrounds an element which includes the background of the element.

Styling Content.

Organising Layouts using Flexbox.

Justify-content:
flex-start, flex-end, centre, space-between, space-around.
Aligns items along x axis.

Align-items:
aligns items along y axis.

Flex direction:
row, column, row-reverse, column reverse

Alters order of items along x or y axis,
reverse changes the order to opposite.
Changing from row to column switches the main axis of justify content/ align items.

Flex-wrap alters whether items will all be squeezed into fit along a single line or whether they will wrap around to fit on multiple lines.

Flex-flow combines flex direction and flex wrap.Flex-flow: column, wrap;

Align content : alters the space in between multiple lines.
